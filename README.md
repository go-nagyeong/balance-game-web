# 🎲 Balance Game Website 🎲
### 회원제의 밸런스 게임 사이트
(21.05.26 ~ 21.05.31)  
&nbsp;

## Description
- 선택하기 힘든 두 선택지 중 하나만을 고르는 밸런스 게임을 구현  
- 밸런스 게임의 질문 구성 및 순서는 고정되며, 총 20개의 질문으로 이루어져 있음  
- 회원가입을 해야만 게임을 할 수 있고, 회원 정보와 게임 결과는 데이터베이스에 저장됨
- 게임을 할 때 저장된 데이터를 이용하여 RECORD 페이지에서 질문 선택 비율을 알 수 있음  
&nbsp;
